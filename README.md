# SWPPP Documentation System

This project contains the interface for the SWPPP document generation system.

## Features

- Project Dashboard with stats
- Detailed Project View with:
  - Location & Environmental Data
  - SWPPP Specifics (BMPs, NOI)
  - Financials (QuickBooks integration)
  - Interactive Map Links
- Status Workflow (New -> Pending -> Processing -> Approved/Manual)

## Setup

1. Run `npm install`
2. Run `npm run dev`
